![ITSM365 Sales Trainer](assets/banner.png)

## 📋 Введение

Тренажёр ITSM365 — это интерактивный инструмент для тренировки навыков продаж. Вы сможете практиковаться в проведении демонстраций, работе с возражениями и закрытии сделок с виртуальным клиентом, который отвечает голосом.

### Что вы сможете тренировать:

- 🎯 **Демонстрации** — проведение презентаций для разных отраслей
- 🛡️ **Работа с возражениями** — ответы на "дорого", "вы competitors", "долго внедрять"
- 💰 **Закрытие сделки** — переход к КП, триалу, следующей встрече
- 📊 **Обоснование бизнес-эффективности** — демонстрация ROI

## 🔑 Получение API ключа (бесплатно)

### Что это за ключ?

Ключ нужен для того, чтобы виртуальный клиент мог отвечать на ваши сообщения с помощью искусственного интеллекта.

### Как получить бесплатно:

1. Перейдите на [console.groq.com](https://console.groq.com)
2. Нажмите **"Sign up"** (регистрация)
3. Заполните форму (имя, email, пароль)
4. Подтвердите email
5. В меню слева выберите **"API Keys"**
6. Нажмите **"Create API Key"**
7. Дайте ключу имя (например, "ITSM365 Trainer")
8. Скопируйте ключ

### Введите ключ в тренажёре:

При первом запуске сценария тренажёр попросит ввести ключ. Вставьте его и нажмите OK.

> Ключ сохранится в вашем браузере, вводить нужно только один раз.

## 🎤 Голосовое взаимодействие

### Включить голосовой ввод:

1. Нажмите кнопку **🎤** в поле ввода сообщения
2. Разрешите доступ к микрофону (требуется только первый раз)
3. Говорите чётко, текст появится автоматически
4. Сообщение отправится после завершения фразы

### Голосовой вывод:

Виртуальный клиент отвечает голосом автоматически. Громкость можно регулировать в настройках вашего устройства.

### Поддержка браузеров:

| Браузер | Голосовой ввод | Голосовой вывод |
|---------|----------------|-----------------|
| ✅ Chrome | Полностью | Полностью |
| ⚠️ Safari | Ограничено | Полностью |
| ❌ Firefox | Нет | Полностью |

> **Рекомендуется Chrome** для полной функциональности.

## 🎯 Как тренироваться эффективно

### Перед тренировкой:

1. **Выберите сценарий** согласно вашим целям
   - Новичок → Начните с "Начальный" уровень
   - Продвинутый → Тренируйтесь на "Продвинутый" сценарии

2. **Подготовьтесь** (5 минут)
   - Вспомните скрипт/structure для этого типа сценария
   - Подумайте о возможных возражениях
   - Поставьте цель (например, "Закрыть на POC")

### Во время тренировки:

3. **Включите голосовой режим** (если в Chrome)
   - Так реалистичнее и полезнее для тренировки

4. **Следуйте структуре**
   - Начните с identifying pains
   - Демонстрируйте value
   - Обрабатывайте возражения
   - Закрывайте на next step

5. **Используйте чат** для подсказок
   - Если не знаете что ответить — посмотрите в notes
   - Виртуальный клиент подскажет если вы "промахнулись"

### После тренировки:

6. **Посмотрите результаты**
   - Ваша оценка из 100
   - Что сделали хорошо (strengths)
   - На что обратить внимание (improvements)

7. **Повторите** сценарий с низкой оценкой
   - Практика делает совершенным!

## 💡 Советы по тренировке

### Демонстрации:

✅ **Правильно:**
- Начните с вопросов "как сейчас работает?"
- Демонстрируйте features через benefits для клиента
- Используйте industry-specific language

❌ **Ошибки:**
- Сразу начинаете "pitch" без установления контекста
- Говорите о features без связи с business pains
- Используете общий язык без industry specifics

### Возражения:

✅ **Правильно:**
- Признайте возражение ("Я вас понял...")
- Не спорьте напрямую
- Переводите в value proposition
- Закрывайте на следующий шаг

❌ **Ошибки:**
- Спорите ("Нет, не дорого!")
- Обесцениваете возражение
- Не закрываете на действие

### Закрытие:

✅ **Правильно:**
- Подтверждаете interest
- Соглашаете scope
- Устанавливаете timeline
- Чёткий next step с accountability

❌ **Ошибки:**
- Оставляете "я вам.send proposal"
- Нет clear timeline
- Не устанавливаете next step

## 🔧 Технические вопросы

### Тренажёр не работает?

**"Голосовой ввод недоступен"**
- Используйте браузер Chrome для полной функциональности
- Проверьте разрешение на микрофон в настройках браузера

**"Не удалось подключиться к API"**
- Проверьте что API ключ введён правильно
- Убедитесь что ключ активен в console.groq.com
- Проверьте подключение к интернету

**"AI отвечает медленно"**
- Обычно ответ занимает 1-2 секунды
- Проверьте скорость интернета
- Попробуйте обновить страницу

### Конфиденциальность:

- Ваш API ключ сохраняется **только в вашем браузере**
- Диалоги не сохраняются на сервере
- Статистика хранится локально в браузере
